# InfluxDB Integration Examples

This directory contains small standalone examples that show how to send data to and read data from InfluxDB 2.x.

## Projects

* [`influx_cpp_writer`](influx_cpp_writer/) – C++17 application that writes a point to InfluxDB using libcurl.
* [`influx_python_reader`](influx_python_reader/) – Python script that runs a Flux query and prints the resulting rows.

Each project has its own README with build/run instructions.
